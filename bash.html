Dear Bash,

Never in a millions year I thought I would meet you. Then again, I never heard about you. I see that you have me opening up two terminals to upload to a server, you must like a lot of attention. You have a lot commands, but I love every single one of them.

  • You let me edit files : nano
  • You let me create files : touch
  • You let me navigate through the directory: cd ~ - home diretory or cd / - root direcotry
  • You let me create contents of file : mkdir
  • You lists all files in current directory : ls
  • If I get lost you let me see current directory location : pwd
  • You let me connect and upload to the server : ssh user@host

Thank you for making my life easier in the this terminal life. You are really something, call me maybe?

Love, Carolyn Castanon
