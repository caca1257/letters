Dear Git,

Never knew about you! I always knew about GitHub, but not about simply Git. You are very easy to love I think. I am not sure if you love me or understand me, but I want to tell you that I do love you and the reasons why I love you.

Why I love you :

  • I can check on your status : git status
  • I can keep track of you and see that you are save : git add .
  • I can let you know of all my changes of my life : git commit -m "text"
  • You can let me see about all my changes, which is nice : git log
  • Whenever I need to change my comments you let me : git commit
  • I can always open up my commits : git checkout(number of commit) -- index.html
  • I can create a branch with you: git branch (name)
  • I can get into our branch anytime : git checkout (name of branch)
  • My favorite is you let me merge branches: git merge (name of branch)
  • But most importantly you understand when a branch just needs to go : git branch -D (name of branch)

I hope this helps you understand my profound love for you.

Love, Carolyn Castanon
