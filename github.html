Dear GitHub,

I met you once before and I did not find you interesting at all. Now we meet again, and I think I like you, but I am not sure yet. I am learning your langauge of love and your kindness of showing the world my terrible projects. I know if we want to be private I need to pay for you, I am sorry I am a broke college student; but together, this thing we have can last forever...I hope.

How to love you through the terminal

To start the burning fire of love
  • to install git : git config --global user.name "[name]"
Creating a new repository
  • git init [project-name]
  • git add README.md
  • git commit -m "first commit"
  • git remote add origin https://github.com/[username]/[file name].git
  • git push -u origin master
Speaking with Remote Branch
  • git merge [branch name]
  • git push
  • git pull

We will merge together and our love will be so strong it will fill like a pull and I hope through the rough times we don't push each other way. Togther we are one master.

Love, Carolyn Castanon
